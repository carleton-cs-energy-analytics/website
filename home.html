Energy Analytics for Improved Campus Sustainability

Our Project

Currently, Carleton's granular energy data is not being taken advantage of for sustainability improvement. We collected, processed, and eventually visualized the data on a dashboard, so that the facilities can observe data trends and detect anomalies in the buildings in the future.

Infrastructure Overview

Technically, our project mainly consists of 6 parts: a value pipeline that transports real-time values from the devices to the database; decoders that put meanings to the static point definitions; a database that stores the information; an brand-new API design that's powered by boolean logic to perform complicated searches; trends UI that visualizes the data in accessible forms; anomaly alerts that inform our users if anything is wrong in the building.
